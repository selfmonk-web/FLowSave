## The problem

You start an AI conversation and it goes brilliantly. The idea is sharp, the direction is clear. Then you iterate — one message too many — and the thread loses focus. The original clarity is gone, buried under revisions and tangents.

You can't go back. You start over and spend the next twenty minutes trying to recreate something that was already perfect.

**FlowSave fixes this.**

---

## What it does

FlowSave lets you save named checkpoints at any point in an AI conversation — like save slots in a video game. When a conversation drifts too far, you open your best checkpoint and paste it into a new chat in seconds.

- **Manual checkpoints** — you decide when to save, full control
- **Name, rate (1–10), tag and annotate** each checkpoint
- **⭐ Gold** — mark your most important checkpoints
- **Filter** by Gold or tag to find what you need instantly
- **One-click restore prompt** — structured, ready to paste into any new chat
- **Multi-project** — keep checkpoints organised by project
- **100% local** — nothing leaves your browser, no account required

Works with Claude, ChatGPT, Gemini, Perplexity, and any other AI chat interface.

3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked**

## How to use

**Saving a checkpoint**

1. Select any text in your AI chat
2. Right-click → **Save as Checkpoint ✦**
3. Give it a name, a rating, tags, and notes
4. Save

Or open FlowSave from the toolbar and click **+ Save** to paste text manually.

**Restoring a checkpoint**

1. Open FlowSave
2. Navigate to your project
3. Find the checkpoint (filter by Gold or tag)
4. Click it → **📋 Copy restore prompt**
5. Open a new chat → paste → continue

The restore prompt includes the project name, checkpoint name, rating, tags, notes, and the full saved text — everything the AI needs to pick up exactly where you left off.
